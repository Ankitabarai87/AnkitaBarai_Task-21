# 🍔 Food Delivery Website (Omnifood Website)- Task-21

This is a responsive **Food Delivery Website** built using **HTML, CSS, and jQuery**.  
The project showcases a modern UI design with smooth scrolling, animations, and multiple sections like features, meals, cities, testimonials, and pricing plans.

---

## 🚀 Features

- ✅ Fully responsive design (Mobile, Tablet, Desktop)
- ✅ Smooth scrolling using jQuery
- ✅ Attractive UI with sections:
  - Hero Section
  - Features Section
  - Meals Showcase
  - How It Works
  - Cities
  - Testimonials
  - Pricing Plans
  - Contact Form
- ✅ Scroll animations using AOS (Animate On Scroll)
- ✅ Hover effects on images and buttons
- ✅ Fixed background images (parallax effect)

---

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
- **jQuery**
- **AOS Library (Animation)**
- **Ionicons (Icons)**
- **Google Fonts**
